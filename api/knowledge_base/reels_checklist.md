# Instagram 伸びるリール動画のチェックリスト

## 1. 冒頭の引き (Hook)
- 最初の 1.5 秒で「あ、これ気になる」と思わせる。
- 文字入れ：画面中央か少し上に大きなフォントで。
- 動き：いきなりインパクトのある動きや表情から。

## 2. 視認性とテロップ
- テロップの配置は、ボタンやキャプションと重ならない位置。
- フォントの読みやすさ：背景とのコントラスト、枠線、背景。
- 音源のリズムに合わせたカット割。

## 3. 内容の構成
- 結論を最初に言う「PREP法」がリールでも有効。
- 「保存したくなる」ノウハウ、リスト形式。

## 4. インサイト向上施策
- キャプションで「保存して後で見返して」と明記。
- 最後に 2 秒程度の CTA (Call To Action) を入れる。
